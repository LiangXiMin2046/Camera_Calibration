# 相机标定
  最近在学习结构光测量的时候整理了相机标定的知识，因此特地做一记录。代码使用了python和opencv库。
# 文件说明
  `draw文件夹：存储标记好角点信息的图片`
  `images文件夹：棋盘格图片，用于标定`
  `calibration.py ：运行代码，请在命令行下执行 python calibration.py`
  `camera_calibration.pdf：关于相机标定的知识总结`
